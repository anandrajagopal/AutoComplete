    auto-complete

    A very fast autocomplete utility

    How to use:

    SuffixTree suffixTree = new SuffixTree();
    List<String> words = new LinkedList<String>() {
        private static final long serialVersionUID = 1676489769011386326L;

        {
            add("banana");
            add("mango");
            add("minimize");
            add("maximize");
            add("bank");
            add("development you$");
        }
    };
    suffixTree.add(words);

    Then:

    List<Integer> indices = new LinkedList<Integer>();
    suffixTree.find("ban", indices);

    will return [1, 5]

    Performance

    I have tested it with over 100,000 words. Very rarely does the search takes 1 millisecond. Mostly the searches time can only be measured in nanoseconds. This is on a mac osx with 4 gb ram.
